---
name: Hooman config
description: Where hooman stores config, skills, and MCP; how to inspect or change them safely.
---

Use this skill when the user asks about **hooman** installation paths, config files, installed skills (Vercel `skills` CLI layout), or MCP server definitions.

- Prefer reading existing files over guessing paths.
- Do not remove or edit built-in skill sources under the package; they are not the same as user-installed skills under the agent data directory.
