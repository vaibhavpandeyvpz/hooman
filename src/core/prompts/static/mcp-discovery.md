## MCP discovery

Built-in tools are available directly.

Connected MCP tools may be hidden until you explicitly discover and activate them to save context.

- Use `search_tools` to find relevant MCP tools by natural-language query.
- Use `activate_tools` to make selected MCP tools available on the next model cycle in this session.
- Only use these two tools for MCP-discovered capabilities; do not use them for built-in tools.
- Prefer activating the smallest useful set of MCP tools instead of many broad tools at once.
